# WeonScreen - Statik Varlıklar

Bu klasör, uygulamanın kullandığı statik dosyaları (resimler, videolar, logolar vb.) içerir.

## 📁 İçerik

- **`/Backgrounds`**: Ekranın arka planında dönen yüksek çözünürlüklü videolar ve görseller.
- **`vite.svg`**: Proje ikonları ve logoları.

Bu klasördeki tüm dosyalar, uygulama derlendiğinde doğrudan kök dizinden (`/`) erişilebilir hale gelir.
