# WeonScreen - Kaynak Kodları (Frontend)

Bu klasör, Vite tarafından derlenen frontend kaynak dosyalarını içerir.

## 📁 Dosyalar

- **`style.css`**: Uygulamanın temel stil tanımlamaları ve animasyonları.
- **`main.js`**: Frontend mantığı ve veri işleme süreçleri.
- **`javascript.svg`**: Geliştirme sürecinde kullanılan bileşenler veya ikonlar.

*Not: Uygulamanın ana ekran mantığı projenin kök dizinindeki `main.js` dosyasında yer almaktadır. Buradaki dosyalar projenin genişletilebilir yapısını desteklemek amacıyla ayrılmıştır.*
